# 🏃 Gogunbunthree (고군분투3)

끝없이 펼쳐지는 도로 위에서 장애물을 피하고, 코인을 모으며, 다양한 아이템을 활용해 최고 점수를 달성하는 **3D 러너 게임**입니다.

## 🎮 게임 소개

**Gogunbunthree**는 3개의 레인을 오가며 달려가는 모바일 최적화 3D 무한 러너 게임입니다. 스와이프로 캐릭터를 조작하고, 다가오는 차량을 피하면서 코인과 아이템을 수집하세요!

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **레인 기반 이동** | 3개의 레인에서 스와이프로 빠르게 이동 |
| **점프 & 슬라이드** | 장애물 위를 뛰어넘거나 아래로 슬라이드 |
| **장애물 회피** | 레인에 등장하는 차량을 피해 달리기 |
| **코인 수집** | 코인을 모아 점수 획득 |
| **파워업 아이템** | 자석(코인 자동 수집), 스타(무적 10초), 헬리콥터(구출) |
| **캐릭터 커스터마이징** | 모자, 가방, 스웨터 등 다양한 스킨 변경 |
| **글로벌 랭킹** | LootLocker 기반 온라인 리더보드 |
| **업적 시스템** | 마일스톤 달성 시 업적 뱃지 획득 |
| **낮/밤 사이클** | 동적 조명과 헤드라이트 연출 |
| **사운드 시스템** | BGM 및 효과음 관리 |

## 🕹️ 조작법

| 입력 | 동작 |
|------|------|
| ← 스와이프 | 왼쪽 레인으로 이동 |
| → 스와이프 | 오른쪽 레인으로 이동 |
| ↑ 스와이프 | 점프 |
| ↓ 스와이프 | 슬라이드 |

## 🛠️ 기술 스택

- **엔진**: Unity 2023.2.20f1
- **언어**: C#
- **플랫폼**: 모바일 (Android)
- **백엔드**: LootLocker SDK (랭킹, 업적)
- **UI**: Unity UI (UGUI) + TextMesh Pro
- **후처리**: Unity Post-Processing Stack v3.5.0
- **레벨 디자인**: ProBuilder v5.2.4

## 📂 프로젝트 구조

```
Assets/

│   ├── Ranking        # 랭킹 화면
│   └── AchivementAndCustomize  # 업적 & 커스터마이징
├── Script/           # 핵심 C# 스크립트
│   ├── Player/       # 플레이어 로직 & 게임 매니저

│   ├── Helicopter/   # 헬리콥터 메커니즘
│   ├── Hook/         # 훅/로프 메커니즘
│   ├── Ranking/      # LootLocker 연동 & 랭킹
│   ├── Sound/        # BGM & SFX 관리
│   ├── UI/           # UI 컨트롤러
│   └── Database/     # 스킨 데이터베이스
├── Shader/           # 커스텀 셰이더
├── Sound/            # 오디오 파일
└── Store/            # 에셋 스토어 리소스
```

## 🚀 빌드 및 실행

1. **Unity Hub**에서 Unity **2023.2.20f1** 버전을 설치합니다.
2. 이 저장소를 클론합니다:
   ```bash
   git clone https://github.com/beannowbean/Gogunbunthree.git
   ```
3. Unity Hub에서 프로젝트를 열고 `Assets/Scenes/MainMenu.unity` 씬을 실행합니다.
4. Android 빌드: **File > Build Settings > Android**를 선택 후 빌드합니다.
